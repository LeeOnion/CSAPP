### 0 对比

二叉搜索树：对平衡无要求，后果是存在退化为链表的风险。

AVL树：左右子树高度最多差1

完美平衡二叉搜索树：左右子树高度必须为0

红黑树：自平衡的二叉搜索树，黑色完美平衡二叉树

### 1 性质

| 性质                                                 |
| ---------------------------------------------------- |
| 每个节点非红即黑                                     |
| root和叶子为黑                                       |
| 父亲红，两个孩子必须黑                               |
| 任意一结点到每个叶子结点的路径都包含数量相同的黑结点 |
| 节点有黑孩子，则有2个孩子                            |

### 2 结构体

```
// 红黑树的结构体带parent，是考虑到左旋和右旋
typedef struct node {
	int data;
	int blockNum;
	struct node *leftChild, *rightChild, *parent;
    int color;
}
```
